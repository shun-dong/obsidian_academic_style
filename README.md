# Obsidian 学术预览样式说明

这份 CSS 主要针对 **预览模式（markdown-preview-view）**，不影响编辑模式，也不修改字体，只做排版与导出优化。

## 功能一览

1. **标题样式**
   - 所有标题（H1–H6）设为黑色、加粗，仅预览生效。
   - H2–H6 启用自动编号：`1`、`1.1`、`1.1.1` …，适合论文/报告结构。

2. **页面与导出布局**
   - 预览区限制为类似 A4 的宽度（max-width ≈ 19cm），居中显示。

3. **列表样式**
   - 无序列表缩进略缩：
     - 一级：圆点 `•`
     - 二级：短横线 `–`
     - 三级：小圆点 `◦`
   - 有序列表：
     - 一级：默认 1, 2, 3…
     - 二级：`(a) (b) (c)…`
     - 三级：`i. ii. iii. …`
   - 微调列表项间距，使段落更紧凑规范。

4. **任务列表隐藏（预览）**
   - 渲染后的任务列表（带复选框）在预览中完全隐藏。
   - 不影响编辑模式中的 `- [ ]` / `- [x]`。

5. **引用（blockquote）**
   - 移除 Obsidian 默认背景色和大边框。

6. **表格**
   - 表格宽度根据内容自动适应，不再强制占满整行。
   - `width: auto + max-width: 100%`，居中显示，避免太宽。
   - 使用顶部/底部细线和加粗表头，风格简洁，适合论文表格。

7. **代码与脚注**
   - 行内代码：浅背景 + 细阴影，略突出但不过分醒目。
   - 代码块保留正常字号，不额外放大/缩小。
   - 脚注：
     - 行内脚注编号用全角括号包裹：`［1］`。
     - 脚注区去掉分割线，整体稍缩进，字号略小。

---

**使用方式**：将 `latex.css` 放入 Obsidian 主题或 Snippet 中，并在「外观 → CSS 代码片段」中启用即可。
